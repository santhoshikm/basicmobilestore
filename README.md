# Basic Mobile Store

## Overview

Basic Mobile Store is a responsive e-commerce front-end website developed using HTML, CSS, and JavaScript. The project provides a modern and user-friendly platform for exploring smartphones from various brands, comparing specifications, and viewing detailed product information.

The website is designed with a clean interface, responsive layouts, and intuitive navigation to deliver a seamless browsing experience across desktop, tablet, and mobile devices.

## Live Demo

Website: [https://basicmobilestore.vercel.app/](https://basicmobilestore.vercel.app/)

## Features

* Responsive design for all screen sizes
* Modern and clean user interface
* Smartphone product catalog
* Brand showcase section
* Product comparison functionality
* Detailed product information pages
* Contact page for user inquiries
* Interactive navigation and user experience
* Optimized layout and performance

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Vercel (Deployment)


## Pages

### Home

Introduces the platform with featured products, highlights, and navigation to key sections.

### Products

Displays smartphones with images, specifications, pricing, and product details.

### Brands

Presents popular smartphone brands and their featured devices.

### Compare

Allows users to compare smartphone specifications and features.

### Contact

Provides a contact form and information for user communication.

## Objectives

* Demonstrate front-end web development skills using HTML, CSS, and JavaScript.
* Build a responsive and visually appealing e-commerce interface.
* Provide users with an easy way to explore and compare smartphone products.
* Showcase modern UI/UX design principles.

## Future Enhancements

* Product search functionality
* Advanced filtering and sorting options
* Shopping cart integration

* Product reviews and ratings
